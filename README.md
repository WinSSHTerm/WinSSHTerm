# WinSSHTerm
A tabbed SSH solution for Windows, combining PuTTY, WinSCP and VcXsrv

https://winsshterm.blogspot.com

Migration tool available [here](https://github.com/P-St/Migrate2WinSSHTerm)

Open RDP/VNC/HTTP sessions and more with [Launch Tools](https://github.com/WinSSHTerm/LaunchTools) - WinSSHTerm's [PowerShell](https://github.com/powershell/powershell) integration

There is a [custom PuTTY build](https://github.com/WinSSHTerm/PuTTY-WST) available that has a scroll behaviour similar to Windows Terminal and other features
